# Groupie Tracker

## Project Overview
Groupie Tracker is a web application that consumes an API containing information about bands and artists. The app displays this data through various visualizations such as cards, tables, lists, and charts.

## API Structure
The API is divided into four parts:
- **artists:** Contains details about bands and artists including their names, images, starting year, first album date, and members.
- **locations:** Includes last and upcoming concert locations.
- **dates:** Contains last and upcoming concert dates.
- **relation:** Links the artists, dates, and locations data.

## Features
- Fetches and displays data from the API with different visualization formats.
- Implements event-driven client-server communication for dynamic data updates.
- Handles errors to ensure stable operation of the website and server.
- Backend developed in Go using only standard packages.
- Includes unit tests for functionality verification.

## Technologies Used
- Go (standard library)
- HTML, CSS (for frontend)

## Usage
This project demonstrates data manipulation, JSON handling, client-server interaction, and event visualization in a full-stack web application.
